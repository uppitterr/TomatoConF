Загрузка спецификации из файла программ проектирования САПР типа  "Протеус".
Файл создается в "Протеус" по команде: Сервис/Спецификация/"Full CSV Output"

(полный вывод спецификации в текстовый файл в формате CVS)

При разрещенном значениии "Создавать идентификаторы" , недостающие элементы справочника идентификаторов создаются автоматически.

У нового идентификатора, нужно в ручную определить соответствие из справочника "Номенклатуры".

При разрешенном значении "Создавать номенклатуру" будут также созданы и элементы для справочника "Номенклатура" аналогичные по названию идентификаторам.

Примечание: До тех пор, пока не будет установлено соответствие элементов, их нельзя перенести в накладную проекта.

Алгоритм работы загрузки:

1) Поиск в первой строчке загружаемого файла реквизитов назначеного формата.

2) Определение позиции реквизита относительно  файла загрузки.

3) Заполнение таблицы идентификаторами и значениями получеными из  файла.

4) Создание новых элементов если это разрешено на форме флажками "Создавать...".

5) Заполнение характеристик у элементов справочника "номенклатуры" если такие указаны в формате загрузки.
