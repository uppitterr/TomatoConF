Консоль печати этикеток.

Назначение: печать этикеток по номенклатуре полученой выборкой из справочника, документа или произвольным запросом с возможностью использования редактируемого, непосредственно перед печатью, макета.

Примечание: обработка разрабатывалась под версию платформы 8.1 конфигурации УТ 10.3 с установленной компонентой "1С:Печать штрихкодов".

Каждый макет может быть сохранен на диске в виде отдельного файла. Кроме самого макета в файл сохраняется информация о параметрах макета и алгоритме получения дополнительных полей (параметров).

Номенклатура, которая будет выводится на этикетках расположена в табличной части "Товары" одноименной вкладки.
Вкладка "Настройки" содержит информацию о размерах макета. Вкладка "ДополнительныеПараметры" может содержать исполняемый код для вычисления дополнительных параметров.

Все переменные (или параметры) макета должны размещатся на макете в квадратных скобках, например [Артикул]. При заполнении макета значениями переменные вычисляются по следующему алгоритму:
 - в первую очередь проверяется имеет ли имя переменной вид [ДопПараметр] или [ДопПараметр.ИмяКлюча], то будет выполнена попытка вычисления по указанному пользователем алгоритму;
 - во вторую очередь будет произведена попытка поиска поля табличной части Товары, с именем совпадающим с именем указанной переменной;
 - в третью очередь будет произведена попытка поиска реквизита номенклатуры, с именем совпадающим с именем указанной переменной;
 - в четвертую очередь будет  произведена попытка поиска свойства номенклатуры,  с именем совпадающим с именем указанной переменной;
Если значение переменной вычисленно не будет, то выводится служебный символ. По-умолчанию "<>", но может быть изменен на вкладке "Дополнительные параметры".

В случаях когда требуется вывести на этикетке значение параметра, которое зависит от номенклатуры, но не являющийся реквизитом или свойством номенклатуры, необходимо прописать код вычисляющий данное значение. Если требуется определить значение одного параметра, то в коде должна быть определена переменная "ДопПараметр" с любым типом значения, которое можно привести к типу "Строка". В случае если необходимо определить два или более дополнительных параметров переменная "ДопПараметр" должна иметь тип "Структура". Обращение к значениям из макета выглядит следующим образом: для параметра с типом "Строка" - [ДопПараметр]; для значения с типом "Структура" - [ДопПараметр.ИмяКлюча].

Для удобства на экране макет этикетки по-умолчанию обведен рамкой, которая не выводится на печать. Любые элементы в виде строк или внедренных рисунков расположеных на макете вне рамки, т.е. вне "тела этикетки" выводится на печать не будут. Поэтому, если вы для данного макета не хотите выводить штрихкод, достаточно сдвинуть его за пределы этикетки.

Режим проверки. Данный режим позволяет проверить корректность заполнения макета. Вход и выход из данного режима осуществляется кнопкой "Проверка". При включении данного режима макет этикетки буде заполнен значениями переменных размещенных на этикетке. По-умолчанию значения беруться для первой строки табличной части "Товары". При изменении значения текущей строки значения будут перезаполнены. Выход из режима "Проверка" возвращает макет в то состояние, в которм он был на момент включения режима.

В общем случае табличная часть "Товары" может не содержать в себе ни каких значений кроме значений вычисленных на на вкладке определения дополнительных параметров. В этом случае этикетки будут выводится в том количестве, сколько строк в табличной части.
